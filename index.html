shadcn-inspired

A class-less example inspired by shadcn components.

Please note that not all shadcn components nor features can be implemented only with CSS.
We try our best to get as close as possible to what it has to offer.

Installation

You can include the framework directly via CDN:

Or download the shadcn-inspired-classless.min.css file from this repository and include it locally:

Then, start writing your HTML without worrying about classes!

*Note that in some of the following components, we use icons.
These icons come from Lucid icon pack and are not shipped in the distribution.
